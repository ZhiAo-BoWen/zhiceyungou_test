{{ app_name }} — 基于业务场景的软硬件选型与项目结构生成平台

本系统基于 LangChain Agent 与大模型 API，面向真实业务场景提供一站式方案生成：

  • 支持表单填写或上传业务文档（txt/md/pdf/doc 等）
  • 自动解析非结构化文本为结构化业务表单
  • 生成技术选型表与技术架构图
  • 推荐硬件配置方案并提供多维度可视化分析
  • 输出完整项目目录结构，并支持写入本地工作空间

技术架构

  • 本地 Web 应用（Flask），突破浏览器沙盒限制
  • LangChain Agent 框架驱动大模型分析
  • 默认接入 Qwen3.7-Plus 大模型，支持切换其他有免费额度的模型
